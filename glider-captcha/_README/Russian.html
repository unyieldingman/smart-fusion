Что такое Glider?

Это простой интерактивный проверочный код для сайтов на CMS PHP-Fusion. Он обладает понятным для пользователей интерфейсом. Для успешной проверки необходимо просто сдвинуть ползунок в крайнее правое положение так же, как и на экране блокировки iPhone.

Возможности

  • динамическое изменение хешей
  • уникальные ключи для различных конфигураций сервера
  • уникальные ключи для каждого пользователя
  • легко редактируемые стили ползунка
  • изменяемые параметры отрисовки ползунка

Системные требования

Для сервера

  • PHP-Fusion 7 и новее
  • PHP 5.1.2 и новее

Для клиента

  • включенные cookies
  • поддержка CSS3

Установка

  • скопировать папку glider в includes/captchas
  • перейти в Панель администратора
  • выбрать вкладку Настройки и открыть страницу Безопасность
  • выбрать из выпадающего списка проверочных кодов glider и нажать Сохранить
