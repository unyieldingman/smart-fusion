Installation
If you are reading this guide you downloaded the latest version of Smart System.
If no, use this link: <Download>.

File transferring:
  • • Unpack downloaded archieve.
  • • Go into the <files> folder.
  • • Copy <infusions>, <administration> and <images> folders into the root directory of your PHP-Fusion powered website.
  • • If your version of PHP-Fusion less than 7.01, copy the <includes> folder into the root directory.


Setting CHMOD rights:
Set CHMOD rights on following folders on your site.
  • • (1) <777> - folder <infusions/smart_system/addons>.
  • • (2) <777> - folder<infusions/smart_system/storage>.
  • • (3) <666> - file <infusions/smart_system/storage/addons.xml>.
  • • (4) <777> - folder<infusions>.
  • • (5) <777> - folder <themes>.
  • • (6) <777> - folder <locales>.
  • • (7) <777> - root folder such as <www> or <public_html>.

WARNING!
If php-scripts on your hosting run as a group and have all rights to transfer files within the host place, you can set no CHMOD.

If you are using the server where php-scripts have no group rights, we recomended to set up rights only on (1), (2), (3), (4), (5), (6) items. Setting up rights on the item number (7) is unsafe.
And on the UNIX or Linux powered server <BB-Codes> and <Updates> can not work properly because of the security policy.

Infuse Smart System 3:
  • • Go to <Admin panel> of your site.
  • • Choose the <System Admin> tab.
  • • Open the <Infusions> page.
  • • Choose the <Smart System> from drop-down list and press the <Infuse> button.
  • • Installation completed. You can find the infusion at the <Infusions> tab as <Addons database>.

  • • BB-Codes
  • • Updates
  • • Locales

  • • Installed infusions
  • • Installed panels
  • • Installed themes
  • • Installed locales

  • • Rights of infusions
  • • Rights of panels
  • • Rights of themes
